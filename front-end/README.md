# How to start App

**Clone Github source: duyphamsalto**

```JS
Run: git clone git@github.com:duyphamsalto/training-react.git
Run: training-react
Run: git fetch --all
Run: git pull -r
```

**Step to start App**

```JS
Run: cd front-end
Run: npm install
Run: npm start
```
